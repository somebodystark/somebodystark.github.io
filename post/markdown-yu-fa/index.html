markdown语法

2023-09-14 1 min read

1.标题语法：

单词或短语前面添加井号 (#) 。# 的数量代表了标题的级别。例如，添加三个 # 表示创建一个三级标题

三级标题

注意 # 与文字之间要有空格

2.换行语法：

在一行的末尾添加两个或多个空格，然后按回车键,即可创建一个换行

3.强调语法：

粗体：单词或短语的前后各添加两个星号

这是粗体 文字

斜体：单词或短语前后添加一个星号

这是斜体 文字

粗体斜体共用：在单词或短语的前后各添加三个星号

这是粗斜体文字

4.无序列表

  • first
  • second
  • third

按两次回车退出无序列表

5.分割线语法：
单独一行上使用三个或多个破折号 (---)

6.链接语法：

链接文本放在中括号内，链接地址放在后面的括号中，链接title可选。

语法：超链接显示名

这是链接语法

7.图片语法：

![图片alt](图片链接 "图片title")。

8.转义字符：\

      • 三级标题
